# Resume Builder
# 📄 Resume Builder Web Application

## 🚀 Overview

This is a full-stack Resume Builder web application that allows users to create, edit, save, and export professional resumes.
It provides a live preview with multiple templates and stores data in a MySQL database using a Spring Boot backend.

---

## ✨ Features

* 📝 Create and edit resume details
* 🎨 Multiple resume templates (Classic, Minimal, Bold, Elegant)
* 🎯 Live preview while typing
* 💾 Save resumes to MySQL database
* 📂 Load and edit saved resumes
* ❌ Delete resumes
* 📄 Export resume as PDF
